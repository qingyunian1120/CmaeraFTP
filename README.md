2.gnss采集方面，定位信息每隔5秒回传至顺丰的TCP服务器，地址，端口号6000，传nmea文件（加$设备号）
3.数据传输方面，优先考虑图片回传。按断点续传设置，在网络条件不好时，记录那些图片没有上传至服务器，待网络顺畅后再传至服务器。
